# 기능 목록
- [x] 코치 이름 입력 받기
  - [x] 2~5명이 입력되었는지
  - [x] 이름이 2~4 글자인지

<br/>

- [x] 코치별로 못 먹는 메뉴 입력 받기
  - [x] 같은 메뉴가 입력되었는지 검사 (중복검사)
  - [x] 메뉴의 개수가 0~2개인지 검사 (개수검사)
  - [x] 주어진 메뉴에 있는 메뉴인지 검사 (메뉴검사)

<br/>

- [x] 카테고리 목록 생성
  - [x] 같은 카테고리가 1개 이하로 존재하는지 확인 (한 주에 같은 카테고리는 최대 2번 가능)

<br/>

- [x] 카테고리당 코치별 메뉴 추천
  - [x] 카테고리 받기 
  - [x] 랜덤함수로 메뉴 생성
  - [x] 못 먹는 메뉴인지 확인
  - [x] 이미 있는 메뉴인지 검사 (한 코치는 한 주에 같은 메뉴를 추천할 수 없다)
  - [x] 저장

<br/>

- [x] 출력 후 종료
